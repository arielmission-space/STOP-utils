Welcome to stop-utils's documentation!
======================================

This package provides utilities for Wavefront Error (WFE) analysis using externally simulated errormaps from STOP analysis. 
The main goal is to provide a decomposition into orthonormal polynomials and a set of utilities to visualize and save the results.
